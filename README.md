# chess_engine (in progess)
### Setup:
```
git clone git@github.com:seaner5302/chess_engine.git
cd chess_engine/chess
make
```
### Executing Tests:
After "make" step in Setup you can execute tests.
```
./test_pawn_moves.x
./test_rook_moves.x
```
